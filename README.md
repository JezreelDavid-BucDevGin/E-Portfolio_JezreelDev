# E-Portfolio_JezreelDev
Personal E-Portfolio Website

This project is a personal e-portfolio website designed to showcase my academic background, technical skills, and project experiences. It serves as a centralized platform where visitors can learn more about my journey as a Bachelor of Science in Information Technology (BSIT) graduate, as well as explore the projects I have developed and collaborated on.

The portfolio highlights key information such as educational history, featured works, and contributions to team-based projects. It is designed with a clean and user-friendly interface to effectively present my profile to potential employers, collaborators, and industry professionals.

Key Features:

Personal profile and academic background (BSIT graduate)
Showcase of individual and collaborative projects
Organized project sections with descriptions and details
Responsive and user-friendly design
Centralized platform for professional presentation

Purpose:
This e-portfolio aims to provide a comprehensive overview of my skills, experiences, and accomplishments, helping establish my professional presence and support career opportunities in the IT field.
